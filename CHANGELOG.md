# Changelog

## [0.1.0] - 2026-01-03

Initial release.

### Features
- Real-time hook monitoring for Claude Code sessions
- Web dashboard and TUI for session management
- Transcript sanitization with configurable redaction profiles
- Research-oriented sharing profiles for transcript contribution
- Artifact linking (PRs, commits, repos) to sessions
- Cost tracking and quality scoring
- Process and port scanning
- Audit logging

### Documentation
- Comprehensive docs at `docs/`
- CLI reference, API reference, security guide
- Contribution flow specification

See commit history for detailed changes.
